# 스마트팜 하드웨어 명세서 (Hardware Specifications)

본 문서는 프로젝트에 사용되는 하드웨어 구성 요소와 아두이노(Arduino Uno R4 WiFi) 핀 맵(Pin Map)을 정의합니다. 펌웨어(C++) 코드는 이 핀 맵을 기준으로 작성되어야 합니다.

## 1. 메인 보드

- **Arduino Uno R4 WiFi**: 메인 제어 및 Wi-Fi 통신, MQTT 브로커와의 통신 클라이언트 역할 수행.

## 2. 센서 (Sensors)

| 부품명                 | 측정 데이터                            | 핀 번호 | 데이터 타입  | 비고                                    |
| ---------------------- | -------------------------------------- | ------- | ------------ | --------------------------------------- |
| **DHT22** (또는 DHT11) | 온도, 습도 (`temperature`, `humidity`) | **D2**  | Digital (In) | Adafruit DHT sensor library 사용        |
| **토양 수분 센서**     | 토양 수분 (`soil_moisture`)            | **A0**  | Analog (In)  | 0~1023 측정값을 0~100% 범위로 매핑 필요 |
| **조도 센서 (LDR)**    | 빛의 밝기 (`light_intensity`)          | **A1**  | Analog (In)  | 0~1023 측정값을 조도 값으로 변환 필요   |

## 3. 액추에이터 (Actuators)

웹 대시보드의 명령을 받아 기기를 켜고 끕니다. 스위칭을 위해 3채널 이상의 릴레이(Relay) 모듈을 사용합니다.

| 부품명         | 제어 대상                  | 핀 번호 | 데이터 타입   | 비고                                                           |
| -------------- | -------------------------- | ------- | ------------- | -------------------------------------------------------------- |
| **Relay CH 1** | 식물 생장용 LED (`led_on`) | **D3**  | Digital (Out) | 제어 신호 HIGH/LOW 여부는 릴레이 스펙에 따라 다름              |
| **Relay CH 2** | 환기 팬 (`fan_on`)         | **D4**  | Digital (Out) | 구동 전류가 크므로 아두이노 자체 전원 대신 외부 전원 인가 권장 |
| **Relay CH 3** | 워터 펌프 (`pump_on`)      | **D5**  | Digital (Out) | 팬과 마찬가지로 별도의 전원(어댑터 또는 배터리) 인가 필요      |

## 4. 회로 구성 및 전원 주의사항

1. **모터류 전원 분리**: 워터 펌프와 환기 팬은 가동 시 순간적으로 많은 전류를 소모합니다. 아두이노의 5V 핀에 모터를 직접 연결하면 보드가 리셋되거나 손상될 수 있으므로, **반드시 모터용 외부 전원(예: 9V/12V 어댑터)을 마련하여 릴레이 접점을 통해 제어**해야 합니다.
2. **릴레이 모듈 제어 신호**: 사용하는 릴레이 모듈이 Active HIGH(HIGH 신호일 때 켜짐)인지 Active LOW(LOW 신호일 때 켜짐)인지 확인 후 펌웨어 코드의 핀 출력 상태를 조정해야 합니다.
3. **네트워크 설정**: Uno R4 WiFi 모델은 내부적으로 ESP32-S3 기반의 Wi-Fi를 지원합니다. MQTT 브로커에 접속하기 위해 펌웨어 코드 내에 2.4GHz Wi-Fi 공유기의 SSID와 비밀번호를 기입해야 합니다.

## 5. 라이브러리 의존성 (Libraries)

- `DHT sensor library` by Adafruit
- `ArduinoJson` by Benoit Blanchon (MQTT 페이로드 파싱 및 생성)
- `PubSubClient` (또는 Arduino Uno R4 WiFi 내장 MQTT 클라이언트 호환 라이브러리)
